SELECCIÓN DE CAPACIDADES

A continuación se se explicará cómo seleccionar los valores de las capacidades para un circuito. En este caso se trabajará con el circuito de un mando a distancia para la puerta de un garaje.

 

Esquemático

 

 

 

Los componentes principales de este circuito son:

  • Microcontrolador PIC18LF252-1/SO
  • Transceptor a 868MHz nRF903
  • Protección contra ESD ESDA5V3SC6
  • Regulador de 5V a 3V  MC78LC33NTR
  • Balun para 900Hz 1:4 LDB31900M20C-416

 

 

 

 

 

CLASIFICACIÓN DE CONDENSADORES

Para que un circuito funcione de manera correcta, en ocasiones hay que añadir diversos componentes como condensadores. Según la función que desempeñen podemos clasificarlos mayoritariamente en 3 grupos: Bypass, Bulk y Decouple.

 

CONDENSADORES DE BYPASS

Estos condensadores se encargan de filtrar el posible ruido producido por la fuente de alimentación y por esto serán los más cercanos a esta.
